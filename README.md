# DuckyDiagnostic

A small troubleshooter using the PicoDucky

This is being subbmitted to Hack Club's YSWS [PicoDucky](picoducky.hackclub.com). A PicoDucky is a custon PCB that acts as a HID device and is open soursed on Github [here](https://github.com/Outdatedcandy92/PicoDucky).

## My project

When the USB is plugged in, press the button one to three times to signal the OS. This opens a terminal and runs several commands to gather information about the computer and automatically perform OS-specific troubleshooting steps. 

### Commands to use

#### Windows
|   Win + r
|   wt
|   Enter

#### Mac
|   Command + Space
|   terminal
|   Enter

#### Linux
|   Ctrl + Alt + T
